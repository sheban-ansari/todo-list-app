#  To-Do List App

A fully functional and responsive To-Do List web application built using HTML, CSS, and Vanilla JavaScript. This app allows users to add, update, delete, and mark tasks as complete, with data stored in the browser's Local Storage for persistence.

---

##  Features

*  Add new tasks
*  Mark tasks as complete (click on task)
*  Edit / Update tasks
*  Delete individual tasks
*  Clear all tasks
*  Local Storage support (data persists after refresh)
*  Fully Responsive (Mobile, Tablet, Desktop)
*  Professional and eye-friendly UI
*  Enter key support for quick task addition

---

##  Technologies Used

* HTML5 – Structure
* CSS3 – Styling & Responsive Design
* JavaScript – Functionality & DOM Manipulation
* Local Storage API – Data Persistence


##  How It Works

1. Enter a task in the input field.
2. Click the **Add** button or press **Enter**.
3. The task appears in the list.
4. Click on a task to mark it as complete.
5. Use the **Edit** button to update a task.
6. Use the **Delete** button to remove a task.
7. Use **Clear All** to remove all tasks.
8. Tasks are automatically saved in Local Storage and remain after page refresh.
